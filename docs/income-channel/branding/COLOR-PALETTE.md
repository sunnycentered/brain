# Color Palette — @DigitalNomadTech Channel Branding

## Overview

A professional, tech-forward palette blending the energy of digital nomad life with clean engineering aesthetics. Built for readability in both dark and light Telegram contexts.

## Primary Colors

| Role | Light Mode Hex | Dark Mode Hex | Name | Usage |
|------|---------------|--------------|------|-------|
| **Primary Accent** | `#0EA5E9` | `#38BDF8` | Sky Blue | Links, CTA buttons, bot highlights, @ mentions |
| **Secondary Accent** | `#6366F1` | `#818CF8` | Indigo | Tags, sub-headings, decorative elements |
| **Success / Action** | `#10B981` | `#34D399` | Emerald | Checkmarks, "go live", subscription confirmation |

## Background Colors

| **Overlay** | `#E2E8F0` | `#334155` | Dividers, separators, borders |

## Gradient

Used on channel banner and bot profile image:

```
Linear gradient (135°): #0EA5E9 → #6366F1 → #8B5CF6
Labels: Sky Blue → Indigo → Purple
```

## Usage Rules

1. **Always use dark mode hex when designing for Telegram's native dark theme.** Telegram uses its own background colors; ensure text contrast ratio ≥ 4.5:1 in both modes.
2. **Links and URLs** should be styled with the primary accent (`#0EA5E9` / `#38BDF8`) — do not use default Telegram link blue.
3. **Bot messages** should consistently use the gradient (Sky Blue → Indigo) as a decorative stripe or background bar above text content.
4. **Channel banner** must use the full gradient across the 320×320 square, with white (`#FFFFFF`) overlay text for maximum legibility.
5. **Starter series post covers** should alternate between the three primary accent colors for visual distinction while maintaining palette consistency.
