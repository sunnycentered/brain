## Brand Consistency Checklist

Before publishing anything branded to @DigitalNomadTech, verify:

- [ ] Colors use hex values from COLOR-PALETTE.md exactly
- [ ] Gradient follows 135° angle with `#0EA5E9 → #6366F1 → #8B5CF6` stops
- [ ] Bot/banner icon uses globe + circuit motif (no text on bot profile image)
- [ ] Tone is direct and useful — zero guru/motivational filler
- [ ] Numbers are real with timestamps — no aspirational screenshots
- [ ] Formatting uses Telegram MarkdownV2 (`_italics_`, `*bold*`, `[links](url)`)
- [ ] Maximum 2 emojis per line, used only as section headers
- [ ] Hashtags follow the established set: `#DigitalNomad #PassiveIncome #RemoteWork #TechTools #SideHustle`
