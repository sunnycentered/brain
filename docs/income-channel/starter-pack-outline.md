## Notion Template Structure

### Linked Database 1: Tool Tracker
| Column | Type | Purpose |
|--------|------|---------|
| Tool Name | Title | Name of the tool |
| Category | Select | Connectivity, Hardware, Software, Finance, Security |
| Status | Status | Recommended, Trying, Testing, Dropped |
| Monthly Cost | Number | Cost in USD |
| Affiliate Rate | Percent | Commission rate earned |
| Review Date | Date | When to re-evaluate the tool |
| Link | URL | Official link (affiliate) |
| Notes | Text | Personal review notes |

### Linked Database 2: Budget Tracker
| Column | Type | Purpose |
|--------|------|---------|
| Expense Name | Title | Description of expense |
| Amount | Number | In local currency + USD auto-convert |
| Currency | Select | USD, EUR, GBP, THB, etc. |
| Category | Relation | Links to Tool Tracker categories |
| Date | Date | When incurred |
| Recurring? | Checkbox | Monthly subscription or one-time |

### Linked Database 3: Workflow Dashboard
| Column | Type | Purpose |
|--------|------|---------|
| Task Name | Title | What needs to be done |
| Client / Project | Relation | Links to projects page |
| Status | Status | Not Started, In Progress, Review, Done |
| Due Date | Date | Deadline |
| Priority | Select | High, Medium, Low |
| Time Zone | Text | Working from (city) |
| Related Tools | Multi-select | Which tools are needed for this task |

## Visual Design Notes for PDF

- **Color palette:** Teal (#2dd4bf) accent, dark slate (#0f172a) headers, white backgrounds
- **Typography:** Inter font family (free, modern), 14pt body, 18pt headings
- **Icons:** Lucide icons for tool categories (WiFi, Laptop, Wallet, Shield, etc.)
- **Screenshots:** Tool dashboards annotated with arrows and callouts
- **Tables:** Clean bordered tables with teal header backgrounds
- **Page count target:** 30–35 pages (content-heavy but scannable)
